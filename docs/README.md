## Quick Reference

### Main Menu Shortcuts

| Key | Action       | Description                             |
| --- | ------------ | --------------------------------------- |
| `n` | New Prompt   | Create a new prompt using any technique |
| `m` | Combine      | Chain multiple techniques together      |
| `t` | Templates    | Use custom YAML templates               |
| `h` | History      | Browse recent prompts                   |
| `f` | Favorites    | View favorite prompts                   |
| `s` | Search       | Search saved prompts                    |
| `p` | Preview Mode | Toggle live preview                     |
| `a` | AI Features  | Access AI-powered features              |
| `c` | Settings     | Configure API keys                      |
| `q` | Quit         | Exit the application                    |

### AI Features Menu

| Key | Feature   | Description                       |
| --- | --------- | --------------------------------- |
| `g` | Generate  | Create prompts from plain English |
| `o` | Optimize  | AI-powered prompt improvement     |
| `t` | Test      | Test prompts across models        |
| `c` | Chains    | Multi-step prompt workflows       |
| `s` | Share     | Export/import libraries           |
| `a` | Analytics | View usage statistics             |
